# Dev Salary Prediction Model

## Description

This model predicts developer salary based on the following parameters:
* Country.
* Developer type.
* Education level.
* Years of experience.
* Employment type.

## Data source

The dataset comes from 2025's [Stack Overflow Annual Developer Survey](https://survey.stackoverflow.co/).
